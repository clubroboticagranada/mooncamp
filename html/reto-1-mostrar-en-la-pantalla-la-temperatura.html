Reto 1. Mostrar en la pantalla la temperatura

Reto 1. Mostrar temperatura en pantalla OLED

Reto 1. Visualización de la temperatura en la pantalla OLED
Objetivo
Desarrollar un programa en MicroBlocks que permita mostrar en la pantalla OLED la temperatura medida por los sensores situados en el interior y en el exterior del invernadero. De esta forma, el usuario podrá conocer en tiempo real las condiciones térmicas del sistema y comparar ambos valores.

Descripción del reto
El invernadero dispone de dos sensores de temperatura:

  • Sensor de temperatura interior: encargado de medir la temperatura dentro de la estructura del invernadero (DHT22).
  • Sensor de temperatura exterior: encargado de medir la temperatura del entorno que rodea al invernadero (El propio que incorpora la placa)
    El programa deberá leer periódicamente los valores proporcionados por ambos sensores y mostrarlos de forma clara y organizada en la pantalla OLED.

Requisitos
El programa debe:

  • Inicializar correctamente la pantalla OLED.
  • Leer la temperatura interior.
  • Leer la temperatura exterior.
  • Mostrar ambos valores en la pantalla identificando claramente cada uno de ellos.
  • Actualizar la información cada segundo.

Ejemplo de visualización
INVERNADERO LUNAR

Interior: 23.5 °C
Exterior: 18.2 °C

Ampliación opcional
Una vez completado el reto básico, puedes incorporar alguna de las siguientes mejoras:

  • Mostrar la diferencia de temperatura entre el interior y el exterior.
  • Incluir iconos o gráficos en la pantalla OLED.
  • Cambiar el mensaje mostrado cuando la temperatura interior supere un valor determinado.
  • Añadir la fecha y hora si el sistema dispone de un reloj en tiempo real usando NTP.

Competencias desarrolladas

  • Programación mediante bloques con MicroBlocks.
  • Lectura y tratamiento de datos procedentes de sensores.
  • Uso de pantallas OLED para la presentación de información.
  • Diseño de interfaces sencillas para sistemas embebidos.
  • Análisis de variables ambientales en sistemas automatizados.
