Reto 4. Datalogger temperatura y humedad

Reto 4. Datalogger temperatura y humedad


Objetivo
Desarrollar un programa en MicroBlocks que permita mostrar las variables de humedad y temperatura gráficamente en modo gráfico contínuo, a modo de data logger. Asimismo, debe permitir grabar en archivo CSV los datos adquiridos.

Descripción del reto
El invernadero dispone de  una serie de sensores, de los cuales deseamos tener un registro contínuo y guardar los datos para un análisis posterior.

Las variables que vamos a graficar en este reto son la humedad interior y la humedad, ambas medidas con el sensor DHT22.

Funcionamiento 

Para este reto, el invernadero debe estar conectado al ordenador y con la aplicación de microBlocks abierto con la placa conectada.

Utilizaremos el icono de mostrar/ocultar gráfico y desde esa misma gráfica veremos como exportar los datos a un fichero csv (Botón derecho del ratón)

En el gráfico aparecerán las variables que hayamos incluido con la función grafico que está en el bloque de menú Salida

Ampliación opcional
Una vez completado el reto básico, puedes incorporar alguna de las siguientes mejoras:

  • Mandar los valores de las variables a una hoja de cálculo usando Http Client.
  • Graficado en tiempo real en la hoja de cálculo.
  • Enviar los datos a una base de datos.
  • Añadirle hora de cada medida empleando la URL https://time.now/developer/api/timezone/Europe/Madrid. Hay que procesar la cadena JSON obtenida.

Competencias desarrolladas

  • Programación mediante bloques con MicroBlocks.
  • Gestión de eventos mediante pulsadores.
  • Control de actuadores en sistemas embebidos.
  • Diseño de interfaces de usuario para dispositivos electrónicos.
  • Implementación de modos manuales y automáticos en sistemas automatizados.
